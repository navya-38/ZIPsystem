1.Login to ZIP with valid credentials
2.Navigate to 3222 interface select customer 
3.Add a policy with LOB=057, SBU=SP,PolicyType=AH,TargetMarket=VIC.
4.Select location from the maintain policy screen press F9 to add location 
5.Select OOC code from drop down in country and address should appear as Out of Canada - A&H only
6.When OOC is added Province and postal code fields are not required and Cresta Zone, Winstorm Zone and Flood Zone will be blank
7.select Accident and Sickness Module for the added location
8.Add the covergaes under Accident and Sickness module
9.Quote and Bind to generate the policy without any tax for OOC location
